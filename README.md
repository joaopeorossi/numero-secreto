# 🎯 Jogo do Número Secreto

Um jogo de adivinhação interativo desenvolvido para praticar lógica de programação e versionamento de código.

Este repositório foi criado como parte dos estudos práticos da Alura, com o objetivo principal de explorar e consolidar os comandos e fluxos de trabalho do Git e do GitHub.

---

### 💻 Tecnologias Utilizadas

O projeto foi construído utilizando as bases da web:
* **HTML5:** Estrutura da página.
* **CSS3:** Estilização e layout.
* **JavaScript:** Lógica do jogo, manipulação do DOM e geração de números aleatórios.

---

### 🧠 O que foi praticado

* **Lógica de Programação:** Condicionais (`if/else`), loops e funções em JavaScript.
* **Versionamento:** Commits, branches e boas práticas de documentação em Markdown.
